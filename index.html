Min3D

A project to explore 3d rendering in javascript using the cpu only (*). Generally speaking all of this is an exceptionally bad idea, but it's fun. Probably.
  • Version 1: the bare minimal implementation needed to render a mesh onto the screen.
  • Version 2: some marginal upgrades from version 1.
  • Version 3: some incremental upgrades from version 2 (solid polygons and some light).
Project page
(*) except for those parts under the hood where the canvas implementation uses the gpu, but those factors are not under our control.
